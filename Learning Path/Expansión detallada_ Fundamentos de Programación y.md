# 🧠 Fase 1 — Fundamentos de Programación y Pensamiento Algorítmico

## ✅ Objetivo general

Adquirir una comprensión sólida de cómo resolver problemas de forma lógica y estructurada, mediante pensamiento computacional, formulación de algoritmos y programación en Python, con el fin de sentar las bases para analizar, transformar y modelar datos posteriormente.

---

## 🔹 Nivel 1: Lógica Computacional y Pensamiento Algorítmico

> 🧩 *"Antes de escribir código, aprende a pensar como una máquina."*

### 🎯 Objetivos de Aprendizaje

- Desarrollar habilidades para descomponer problemas.
- Comprender cómo estructuran los algoritmos la solución a un problema.
- Representar algoritmos en forma de pseudocódigo o diagramas de flujo.
- Familiarizarse con la lógica booleana y estructuras de control.

### 🧱 Contenidos

- ¿Qué es un algoritmo? Entrada, proceso, salida.
- Operadores lógicos: AND, OR, NOT.
- Lógica proposicional y tablas de verdad.
- Condicionales: `if`, `else`, `elif`.
- Ciclos: `for`, `while`, control de flujo.
- Pseudocódigo (notación estructurada).
- Diagramas de flujo.
- Introducción a la recursividad.

### 📚 Recursos Sugeridos

- 🎓 Curso gratuito: [CS50x – Semana 0](https://cs50.harvard.edu/x/2025/)
- 📘 Libro: [How to Think Like a Computer Scientist](https://allendowney.github.io/ThinkPython/index.html#) 
- 🧠 Plataforma: [Visualgo.net](https://visualgo.net/en)
- 🧪 Ejercicios prácticos: [Python Tutor](http://pythontutor.com)

### ✅ OKR (Objetivos y Resultados Clave)

| Objetivo                                   | Resultado Clave                                                                 |
|--------------------------------------------|----------------------------------------------------------------------------------|
| Desarrollar pensamiento algorítmico básico | ✔️ Completar al menos 20 ejercicios de lógica (tablas de verdad, bucles, etc.)  |

## 🔹 Nivel 2: Primeros Pasos con Python

> 🐍 *"Elige Python porque te permite pensar en el problema, no en la sintaxis."*

### 🎯 Objetivos de Aprendizaje

- Aprender la sintaxis básica de Python para construir algoritmos.
- Escribir programas simples para resolver tareas cotidianas.
- Utilizar estructuras básicas de datos y control.

### 🧱 Contenidos

- Tipos primitivos: `int`, `float`, `str`, `bool`
- Variables y operadores aritméticos/lógicos
- Estructuras de control:
  - Condicionales (`if`, `elif`, `else`)
  - Bucles (`for`, `while`)
- Funciones: definición, argumentos, `return`
- Colecciones:
  - Listas (`list`)
  - Tuplas (`tuple`)
  - Diccionarios (`dict`)
  - Conjuntos (`set`)
- Introducción a excepciones (`try/except`)
- Entornos de desarrollo: Google Colab, Jupyter, VS Code

### 📚 Recursos Sugeridos

- 📘 Libro: *Python Crash Course* (capítulos 2–8)
- 📙 Curso interactivo: [Automate the Boring Stuff with Python](https://automatetheboringstuff.com/)
- 🎥 YouTube: [Curso Python desde Cero – Píldoras Informáticas](https://www.youtube.com/playlist?list=PLU8oAlHdN5BmpIQGDSHo5e1r4ZYWQ8m4B)

## 🔹 Nivel 3: Estructuras de Datos y Algoritmos Fundamentales

> 🧠 *"Data science sin entender cómo se organiza la información es solo usar librerías sin conciencia."*

### 🎯 Objetivos de Aprendizaje

- Comprender cómo se almacenan y procesan los datos.
- Implementar algoritmos clásicos de ordenamiento y búsqueda.
- Estimar la eficiencia de un algoritmo (noción básica de complejidad).

### 🧱 Contenidos

- **Listas y arreglos**: inserción, eliminación, slicing
- **Diccionarios y conjuntos**: claves, valores, operaciones
- **Pilas (`stack`) y colas (`queue`)**: LIFO y FIFO
- **Algoritmos clásicos**:
  - Búsqueda lineal y binaria
  - Ordenamientos: burbuja, inserción, selección, merge
  - Recursividad básica (factorial, fibonacci)
- Complejidad algorítmica (notación Big O — a nivel intuitivo)

### 📚 Recursos Sugeridos

- 📕 Libro: *Data Structures and Algorithms in Python* (Goodrich)
- 🎥 Curso en español: [Fundamentos de Algoritmos - freeCodeCamp](https://www.youtube.com/watch?v=8hly31xKli0)
- 💻 Práctica: [Exercism.io](https://exercism.io), [LeetCode (Easy)](https://leetcode.com), [HackerRank](https://www.hackerrank.com/skills-directory)

### ✅ OKR (Objetivos y Resultados Clave)

| Objetivo                          | Resultado Clave                                                                 |
|----------------------------------|----------------------------------------------------------------------------------|
| Comprender estructuras dinámicas | ✔️ Implementar 2 estructuras: una pila (stack) y una cola (queue)               |
| Aplicar algoritmos en Python     | ✔️ Programar 3 algoritmos de ordenamiento y 2 de búsqueda                        |
| Analizar eficiencia algorítmica  | ✔️ Comparar tiempos de ejecución básicos usando `time` o `timeit`                |

## 💡 Consejos para Consolidar esta Fase

- 🔁 Practica todos los días: mínimo 30–60 minutos de código.
- 📒 Toma apuntes visuales: diagramas, esquemas y flujos.
- 🧪 Resuelve problemas en papel antes de codificar.
- 🧠 Refuerza con preguntas tipo: “¿Qué pasaría si…?”, “¿Cuál sería la salida si…?”
